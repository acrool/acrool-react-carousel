# im-carousel

## Install

```bash
npm install --save im-carousel
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'im-carousel'
import 'im-carousel/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
